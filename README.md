# Toastmasters VPE Exporter

A browser extension to help Toastmasters VP Education export data for weekly reports.

## Features
- Export DCP Points
- Export Level Awards
- Export Paths in Progress
- Export Agenda
- Export Last Speaker Dates

## Installation
1. Open Chrome/Edge and go to `chrome://extensions`.
2. Enable "Developer mode" (toggle in top right).
3. Click "Load unpacked".
4. Select this folder (`toastmasters-vpe-exporter`).

## Usage
1. Navigate to the Toastmasters website.
2. Click the extension icon.
3. Click the button for the data you want to export.
4. Check the console (F12) for the output (currently placeholders).
